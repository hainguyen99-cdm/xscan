OBS Widget Test Page

Test Widget Endpoint

This page allows you to test the OBS widget endpoint by providing an alert token.

Widget Preview

If the widget loads successfully, you'll see it below. This simulates how it would appear in OBS.

Enter an alert token and click "Test Widget" to see the preview

Test Instructions

  1. Make sure your backend server is running on the specified URL
  2. Enter a valid alert token from your OBS settings
  3. Click "Test Widget" to see the widget in action
  4. Use "Test Token Endpoint" to verify the token is valid
  5. The widget should display with animations and be ready for alerts

Expected Behavior

  • Widget should load with a transparent background
  • Connection status indicator should show connection state
  • Test alert should appear after 2 seconds if connected
  • Widget should be responsive and properly styled
